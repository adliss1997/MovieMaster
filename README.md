# 🎬 MovieMaster

Веб-приложение для поиска и просмотра информации о фильмах с использованием API The Movie Database (TMDB).

## 🚀 Функционал
- 🔍 Поиск фильмов по названию
- 🎭 Фильтрация по жанрам, году выпуска, рейтингу
- ❤️ Добавление фильмов в "Избранное" (сохраняется в localStorage)
- 📄 Детальная страница фильма с трейлером, актёрами, бюджетом
- 📱 Адаптивный дизайн для мобильных устройств

## 📦 Зависимости
Проект не требует установки зависимостей через npm. Все библиотеки подключаются через CDN:
- [Material Icons](https://fonts.google.com/icons) — иконки
- [Google Fonts (Inter)](https://fonts.google.com/specimen/Inter) — шрифт
- TMDB API — источник данных о фильмах

> ⚠️ Для работы приложения необходимо получить бесплатный API-ключ на [themoviedb.org](https://www.themoviedb.org/settings/api) и добавить его в файл `script.js` в объект `TMDB_CONFIG`.
